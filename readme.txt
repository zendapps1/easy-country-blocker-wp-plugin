### Block the trespassers:

In some instances, the online site owner does not want to show the presence of his/her site in a specific country. Keeping this fact in mind, the Easy Country Blocker Plugin has been produced by a range of expert plugin developers. The plugin is an expert in blocking unwanted countries where the site owner doesn’t want his/her site to be exposed to a specific location and also suspected IP range. Easy Country Blocker helps the proprietor to block all undesired from his/her site.   

For instance, if an online retailer is operating in his/her domestic country only, and getting orders from a country in which it doesn't have business-related access then he/she can use Easy Country Blocker to bar all the nations where he/she doesn’t want to conduct his business.

== Installation ==

= Option 1. =

- Upload the entire `easy-country-blocker` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Easy Country Blocker" —> Click install.
- After plugin is downloaded, click active plugin
